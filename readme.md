
## Bot 靜香 v1.0
A LineBot for Cixin Upper School Students and Teachers. Based on Laravel 5.3.

### Installation
```zsh
$ git clone https://github.com/iscixin/LaravelBot MyProject
$ cd MyProject
$ cp .env.example .env
$ php artisan key:generate
$ composer install
$ vi .env
 
 GoogleAPIKey=
 LineBotchannelId=
 LineBotchannelSecret=
 LineBotchannelAccessToken=
```
